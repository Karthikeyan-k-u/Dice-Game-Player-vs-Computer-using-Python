# 🎲 Dice Game: Player vs Computer

## 📌 Overview

This is a simple **command-line dice game** built using Python where a player competes against the computer. The game rolls dice values, determines the winner, and stores the results.

---

## 🚀 Features

* 🎯 Player vs Computer gameplay
* 🎲 Random dice generation
* 🏆 Winner determination
* 📊 Score history tracking
* 🗑️ Clear score history option

---

## 📂 Project Structure

```
project/
│
├── main.py             # Main game logic
├── dice_engine.py      # Dice rolling & winner logic
├── score_manager.py    # Score storage & management
└── dice_scores.txt     # Score history file
```

---

## ⚙️ How It Works

* Player enters a dice value (1–6)
* Computer generates a random value
* Winner is decided based on higher value
* Results are saved in a file
* Scores can be viewed or cleared

---

## ▶️ How to Run

1. Install Python
2. Save all files in the same folder
3. Run:

```
python main.py
```

---

## 🖥️ Menu Options

```
1. Play Game
2. View Score History
3. Clear Score History
4. Exit
```
